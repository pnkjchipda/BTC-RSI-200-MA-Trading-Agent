How execution works

On every new bar the engine runs the full loop:

  1. Perceive — ingest OHLC + recompute RSI(14) / MA(200)
  2. Decide — apply BUY / SELL / HOLD rule
  3. Execute — emit MARKET_BUY or MARKET_SELL, place protective STOP_LOSS + TAKE_PROFIT, ratchet trailing stop on new highs
  4. Manage risk — check stop / TP / trailing fill on every bar's high-low range, prioritised stop → TP → signal

The executor is exchange-agnostic. The current build ships with a paper broker (deterministic fills at the simulated price). Swapping in Bitget, Binance, or any CCXT-supported venue is a single adapter.

// Pluggable broker interface (paper today, live tomorrow)
class BitgetBroker {
  constructor(apiKey, secret, passphrase) { /* sign requests */ }

  async marketBuy(symbol, qty) {
    // POST /api/v2/spot/trade/place-order
    return fetch('https://api.bitget.com/api/v2/spot/trade/place-order', {
      method:'POST',
      headers: this.sign({ symbol, side:'buy', orderType:'market', size:qty }),
      body: JSON.stringify({ symbol, side:'buy', orderType:'market', size:qty })
    }).then(r => r.json());
  }

  async placeStop(symbol, qty, stopPrice) {
    // POST /api/v2/spot/trade/place-plan-order
    return this.signedPost('/place-plan-order', {
      symbol, side:'sell', triggerPrice:stopPrice, size:qty,
      orderType:'market', planType:'normal_plan'
    });
  }

  async placeTakeProfit(symbol, qty, tpPrice) {
    return this.signedPost('/place-plan-order', {
      symbol, side:'sell', triggerPrice:tpPrice, size:qty,
      orderType:'market', planType:'normal_plan'
    });
  }
}

// The strategy engine never changes — only the broker swaps.

About this Agent

This is an autonomous Bitcoin trading agent that combines a momentum filter (200-day Moving Average) with a contrarian timing signal (RSI 14). It only buys oversold dips inside a confirmed uptrend, and it always defines its exit before it enters — a hard stop loss, a take-profit target, and a ratcheting trailing stop. Every trade risks no more than 2% of the account, and every decision is explained in plain English.

Data source: Binance spot BTCUSDT daily klines. Indicators: pandas-ta (RSI, SMA). Backtest: walk-forward, no look-ahead, exit priority = stop → take-profit → signal.
